/send/
  • Input
    • ?name=: Tên người gửi (0 < name.length <=100)
    • &content=: Nội dung gửi (0 < content.length <=4000)
  • Return
      • Thành công
        • sent: true
        • timestamp: Thời gian đã gửi (Dưới dạng xâu DD/MM/YYYY HH/MM/SS)
      • Thất bại
        • sent: false
/get/
  • Input


  • Return
      • Thành công
        • got: true
        • data: Tất cả các tin nhắn được lưu trữ dưới dạng JSON Dưới đây chỉ là đại diện cho 1 tin nhắn (Object)
          • name: Người đã gửi
          • content: Nội dung
          • timestamp: Thời gian đã gửi (Dưới dạng xâu DD/MM/YYYY HH/MM/SS)
      • Thất bại
        • got: false
